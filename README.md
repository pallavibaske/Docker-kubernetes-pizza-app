# 🍕 Docker Kubernetes Pizza App

## Project Overview
This project demonstrates the deployment of a Cloud Pizza application using Docker, Kubernetes, and AWS Cloud services. The application is containerized with Docker, deployed on Kubernetes, and integrated with AWS services for cloud infrastructure, storage, monitoring, and security.

## Technologies Used
- Docker
- Kubernetes
- Amazon EC2
- Amazon S3
- AWS Lambda
- Amazon DynamoDB
- AWS IAM
- Amazon CloudWatch
- Amazon VPC

## Features
- Containerized the application using Docker.
- Deployed the application using Kubernetes.
- Hosted the application on Amazon EC2.
- Configured Amazon S3 for storage.
- Used AWS Lambda for serverless processing.
- Stored application data in DynamoDB.
- Managed user access using AWS IAM.
- Monitored application performance using Amazon CloudWatch.
- Configured networking using Amazon VPC.

## Architecture
Docker → Kubernetes → Amazon EC2 → S3 → DynamoDB → Lambda → CloudWatch

## Learning Outcomes
- Docker containerization
- Kubernetes deployment
- AWS Cloud deployment
- Infrastructure management
- Monitoring and security implementation
